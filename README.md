# tgcofficial.github.io


First clone the project using command line

```bash
git clone https://github.com/tgcofficial/tgcofficial.github.io.git
```

or if you have GitHub CLI

```bash
gh repo clone tgcofficial/tgcofficial.github.io
```

## Host the website
